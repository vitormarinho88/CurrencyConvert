## 🚀 Sobre o Projeto

O **Currency - Convert** foi criado com o objetivo de praticar conceitos essenciais do desenvolvimento Front-End moderno, manipulação do DOM  utilizando JavaScript.

Além da funcionalidade prática de conversão monetária, o projeto também reforça conceitos de responsividade, experiência do usuário e organização de código.
